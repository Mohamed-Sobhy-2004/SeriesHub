**SeriesHub** is a **dynamic TV shows explorer** built with **HTML, CSS, and Vanilla JavaScript**.  
It fetches real data from the [TVMaze API](https://www.tvmaze.com/api) and displays it in a clean, modern dashboard with interactive features.

---

## 🚀 Features

- **Live Search** – Search for your favorite TV shows with instant results.  
- **Responsive Design** – Fully compatible with desktop and mobile devices.  
- **Dark / Light Mode** – User-friendly theme toggle with LocalStorage support.  
- **Dynamic Cards** – Each show displays an image, name, rating, language, and runtime.  
- **Loading Skeleton** – Smooth loading animation while fetching data.  
- **Clean UI / UX** – Modern layout with hover effects for an engaging experience.
